# 🚀 Task 1 — Install and Configure Sliver C2

## 🎯 Objective

Install the Sliver framework and establish a working lab environment.

### 🔹 Update System

```bash
sudo apt update && sudo apt upgrade -y
```

### 🔹 Install Dependencies

```bash
sudo apt install -y git wget curl build-essential
```

### 🔹 Download Sliver

```bash
cd /opt

sudo wget https://github.com/BishopFox/sliver/releases/download/v1.5.42/sliver-server_linux -O sliver-server

sudo wget https://github.com/BishopFox/sliver/releases/download/v1.5.42/sliver-client_linux -O sliver-client
```

### 🔹 Make Executable

```bash
sudo chmod +x sliver-server sliver-client
```

### 🔹 Create Symbolic Links

```bash
sudo ln -s /opt/sliver-server /usr/local/bin/sliver-server

sudo ln -s /opt/sliver-client /usr/local/bin/sliver-client
```
